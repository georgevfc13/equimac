📊 MAPEO DE FUNCIONALIDADES
═══════════════════════════════════════════════════════════════════════════════

CREAR PRODUCTO:
  ├─ Usuario abre: http://localhost/equimac/?accion=formulario
  ├─ Ve: views/formulario.php
  ├─ Envía: POST a index.php?accion=guardar
  ├─ Procesa: InventarioController::guardar()
  ├─ Valida: validar() en controlador
  ├─ BD: Inventario::crear() en modelo
  └─ Muestra: views/lista.php actualizado

LISTAR PRODUCTOS:
  ├─ Usuario abre: http://localhost/equimac/
  ├─ Ejecuta: InventarioController::listar()
  ├─ Obtiene: Inventario::obtenerTodos() del modelo
  ├─ Muestra: views/lista.php
  └─ Incluye: Estadísticas, búsqueda, tabla

EDITAR PRODUCTO:
  ├─ Usuario hace clic: "✎ Editar" en tabla
  ├─ Va a: http://localhost/equimac/?accion=formulario&id=1
  ├─ Carga: Inventario::obtenerPorId($id)
  ├─ Muestra: formulario.php pre-cargado
  ├─ Envía: POST a index.php?accion=guardar
  ├─ BD: Inventario::actualizar($id, $datos)
  └─ Resultado: Lista.php actualizado

ELIMINAR PRODUCTO:
  ├─ Usuario hace clic: "🗑 Eliminar"
  ├─ Confirma: JavaScript alert
  ├─ Va a: http://localhost/equimac/?accion=eliminar&id=1
  ├─ Procesa: InventarioController::eliminar()
  ├─ BD: Inventario::eliminar($id)
  └─ Muestra: lista.php sin el producto

BUSCAR PRODUCTO:
  ├─ Usuario escribe en buscador
  ├─ Se ejecuta: JavaScript en tiempo real
  ├─ Filtra: Tabla localmente
  ├─ O envía: GET con parámetro ?buscar=término
  ├─ Procesa: InventarioController::listar()
  ├─ BD: Inventario::obtenerTodos($filtro)
  └─ Muestra: Tabla filtrada

🔄 FLUJO DE DATOS
═══════════════════════════════════════════════════════════════════════════════

HTTP REQUEST
    ↓
index.php (ENRUTADOR)
    ↓
Identifica ?accion= (listar|formulario|guardar|eliminar)
    ↓
Instancia InventarioController
    ↓
Ejecuta método correspondiente
    ↓
Controlador obtiene parámetros (GET/POST)
    ↓
Valida entrada (client + server)
    ↓
Llama métodos de Inventario (Modelo)
    ↓
Modelo ejecuta queries a MySQLi
    ↓
Base de datos procesa consulta
    ↓
Retorna datos a Modelo
    ↓
Modelo retorna a Controlador
    ↓
Controlador prepara datos para vista
    ↓
Renderiza views/layout.php (HTML base)
    ↓
Incluye views/lista.php, formulario.php, etc
    ↓
Incluye public/css/style.css (estilos)
    ↓
Incluye public/js/script.js (interacción)
    ↓
HTTP RESPONSE (HTML completo)
    ↓
NAVEGADOR renderiza página
