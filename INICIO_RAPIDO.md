## ⚡ 5 PASOS PARA COMENZAR

### 1️⃣ Iniciar XAMPP
```bash
Abre: C:\xampp\xampp-control.exe
Click "Start" → Apache
Click "Start" → MySQL
Espera a que digan "Running"
```

### 2️⃣ Crear Base de Datos
```bash
Abre navegador: http://localhost/phpmyadmin
Crear BD: "equimac"
Copiar: C:\xampp\htdocs\equimac\database.sql
Pegar en SQL de phpMyAdmin
Click "Ejecutar"
```

### 3️⃣ Verificar el Sistema
```bash
Navegador: http://localhost/equimac/diagnostico.php
Confirma que TODO muestre ✓
```

### 4️⃣ Usar el Sistema
```bash
Navegador: http://localhost/equimac
¡Comienza a gestionar tu inventario!
```

### 5️⃣ (Opcional) Validar Todo
```bash
Navegador: http://localhost/equimac/test.php
Ver reporte completo de validación
```
