📋 ¿QUÉ ES EQUIMAC?
═══════════════════════════════════════════════════════════════════════════════

EQUIMAC es un **Sistema Profesional de Gestión de Inventario** desarrollado
con arquitectura **MVC** (Model-View-Controller) usando:

  🔵 PHP Nativo      - Backend simple y poderoso
  🟩 MySQL           - Base de datos local
  🟪 HTML5 + CSS3    - Interfaz moderna
  🟨 JavaScript      - Interactividad

✨ CARACTERÍSTICAS PRINCIPALES:
  ✓ CRUD Completo        (Crear, Leer, Actualizar, Eliminar)
  ✓ Búsqueda Inteligente  (Por código, nombre, categoría)
  ✓ Validaciones         (Cliente + Servidor)
  ✓ Seguridad Integrada   (Prepared statements, escapado HTML)
  ✓ Diseño Responsivo    (Funciona en desktop, tablet, móvil)
  ✓ 100% Funcional       (Listo para usar)

⚡ INICIO RÁPIDO (4 PASOS)
═══════════════════════════════════════════════════════════════════════════════

PASO 1: INICIAR XAMPP
  ─────────────────────────────────────────────────────
  1. Abre XAMPP Control Panel
  2. Haz clic en "Start" para Apache
  3. Haz clic en "Start" para MySQL
  4. Espera a que diga "Running"

  ✅ Listo: Servers iniciados


PASO 2: CREAR BASE DE DATOS
  ─────────────────────────────────────────────────────
  1. Abre navegador
  2. Ve a: http://localhost/phpmyadmin
  3. Crea nueva BD llamada "equimac"
  4. Copia el contenido de 📄 database.sql
  5. Pégalo en phpMyAdmin → SQL
  6. Haz clic en "Ejecutar"

  ✅ Listo: Base de datos creada


PASO 3: VERIFICAR INSTALACIÓN
  ─────────────────────────────────────────────────────
  1. Abre navegador
  2. Ve a: http://localhost/equimac/diagnostico.php
  3. Confirma que todo muestra ✓ (check marks)

  ✅ Listo: Sistema verificado


PASO 4: USAR EL SISTEMA
  ─────────────────────────────────────────────────────
  1. Abre navegador
  2. Ve a: http://localhost/equimac
  3. ¡Comienza a gestionar tu inventario!

  ✅ Listo: ¡Sistema funcionando!

═══════════════════════════════════════════════════════════════════════════════
🎮 CÓMO USAR (Lo Básico)
═══════════════════════════════════════════════════════════════════════════════

CREAR PRODUCTO:
  1. Click en "+ Nuevo Producto" (azul)
  2. Completa el formulario
  3. Click en "✓ Crear Producto"

EDITAR PRODUCTO:
  1. En tabla, encuentra el producto
  2. Click en "✎ Editar"
  3. Modifica los datos
  4. Click en "✓ Guardar Cambios"

ELIMINAR PRODUCTO:
  1. En tabla, encuentra el producto
  2. Click en "🗑 Eliminar"
  3. Confirma la eliminación

BUSCAR PRODUCTO:
  1. En buscador, escribe código, nombre o categoría
  2. Tabla se filtra automáticamente

🆘 PROBLEMAS COMUNES
═══════════════════════════════════════════════════════════════════════════════

❌ "Página en blanco"
   ✓ Abre la consola (F12)
   ✓ Busca errores en "Console"
   ✓ Lee README.md sección "Solución de problemas"

❌ "Error: Conexión a BD rechazada"
   ✓ Verifica que MySQL esté iniciado
   ✓ Verifica que database.sql fue ejecutado
   ✓ Abre: http://localhost/equimac/diagnostico.php

❌ "Caracteres extraños (acentos)"
   ✓ Charset está en UTF-8 (ya configurado)
   ✓ No debería haber problema

❌ "Tabla de productos vacía"
   ✓ Ejecuta database.sql en phpMyAdmin
   ✓ Verifica que data se agregó
